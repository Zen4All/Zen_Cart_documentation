What's Upcoming ... Changes from prior versions

The fixes/changes targeted for v2.1.x include:

  • Core: Shipping modules now inherit from ZenShipping base class to reduce code duplication and ensure consistency.


The fixes/changes targeted for v2.0.1 include:

  • Admin: NOTIFY_ADMIN_CUSTOMERS_LISTING_ELEMENT notifier enhanced to pass entire customer query result.
  • Core: Many PHP8 logs have been fixed. If you are getting debug logs, please apply this update..
  • Storefront: New Products listing now shows new products as configured.
  • Storefront: Shopping Cart class in_cart_check() function fixed.
